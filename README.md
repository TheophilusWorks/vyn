# vyn

> A simple multi-purpose bot that is both scalable and easy to use. It is built using my [@theophilusdev/conduit](https://npmjs.com/@theophilusdev/conduit) wrapper library and is designed to be modular, allowing for easy customization and expansion.

---

No tutorials for now, this is a semi-private repository that I use to test out new features and ideas for my wrapper library. However, if you're interested in learning how to build a bot like this, feel free to check out the source code and experiment with it yourself!
